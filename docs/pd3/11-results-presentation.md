## Large-scale test: 500 independent voters

One test does not reuse Hardhat's 20 default accounts — it **generates 500 brand-new wallets**, registers all of them, runs 6 proposals, and casts 500 votes with a deliberately uneven distribution.

- Contract's on-chain tally is cross-checked against an independently tracked JS-side count — for **every** proposal, not just the winner
- Vote gas cost stays flat regardless of how many voters are registered (~71k–105k gas per vote, same at 3 voters or 2,000)
- Runs in ~4 seconds as part of the normal test suite; `VOTASJ_SCALE_VOTERS=2000 npm test` pushes it further (~16s)

## End-to-end demo: it runs, and it checks itself

`npm run demo` deploys both contracts, runs a full citizen cycle, **and actively tries 5 illegal actions**:

| # | Illegal action attempted | Result |
| - | ------------------------- | ------ |
| 1 | Register the same voter twice | ⛔ Reverted — `"VoterRegistry: already registered"` |
| 2 | Unregistered address submits a proposal | ⛔ Reverted — `"Budget: submitter not registered"` |
| 3 | Unregistered address votes | ⛔ Reverted — `"Budget: voter not registered"` |
| 4 | Same voter votes twice | ⛔ Reverted — `"Budget: already voted"` |
| 5 | Close the cycle before its deadline | ⛔ Reverted — `"Budget: cycle still running"` |

If any of these had _not_ reverted, the script itself would exit non-zero.

## CI pipeline: the numbers are checked, not just reported

Every push to `main` and every pull request runs:

- **Lint** — solhint + eslint + markdownlint, on **Ubuntu and Windows**
- **Test** — 43 tests + gas report, on **Ubuntu and Windows** × **Node 20 and 22** (4 combinations), plus a 1000-voter re-run of the scale test
- **Coverage** — **hard gate at 100%** — the build fails, not just warns, if any category drops; badge auto-updates on `main`
- **Metrics** — structural report artifact
- **Demo** — the full end-to-end run + guard-rail assertions, on **Ubuntu and Windows**, artifact uploaded per OS

Anyone can open the GitHub Actions tab and download any of these reports without cloning the repository.
